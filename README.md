scripter
========

a paltformer.

more to come.

###how to play:
the following commands are supported:

command | usage | function
--------|-------|-----------
`walk <direction:l/r> <steps:integer>` |`walk r 10`| walk 10 steps right
`jump`| `jump`| jump
`stop`|`stop`| stop whatever you're doing.
